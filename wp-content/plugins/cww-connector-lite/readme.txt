== Description ==

CWW Connector Lite is an addon for contact form 7 which allows you to collect leads from contact form 7 to ActiveCampaign. 
This simple yet powerful plugin sends contact form 7 data to ActiveCampaign. 
The plugin is straightforward to use, also there is detailed documentation to set it up.
